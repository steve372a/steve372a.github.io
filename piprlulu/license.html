🐾 Piprlulu 家族：使用与创作小贴士！ (Community Guidelines)

很高兴你喜欢 Piprlulu 家族！为了保护点点、皮皮和露露的成长，在开始你的创作或使用代码库之前，请花一点点时间阅读以下规则：

1. 认识我们的资产

  • 什么是 Piprlulu 资源？
    包括所有相关的图像（.png, .svg, .ico）、模型、音视频、角色设定，以及带有我们视觉标识的代码。
  • 关于授权：
    也许代码可能在 MIT 协议下，但 Piprlulu 的所有形象资产是不受 MIT 协议约束的。它们的著作权及最终解释权是属于 Sanakaprix (steve372a) 的。它们是受到保护的。

2. 欢迎你的二次创作（非商用）

我们非常鼓励大家发挥创意！只要是非营利性质，你就可以：

  • 绘制插画、漫画，制作同人视频或撰写文字故事。
  • 制作自己使用的实体小物件，完全可以送人哦！但是千万不要大规模地卖哦ヾ(≧▽≦*)o。
  • 在社交媒体分享你的作品（建议署名 steve372a 或 Sanakaprix，我们会很开心的！ʕ•ᴥ•ʔ）。
  • 注意： 千万不能冒充原作者，更不要把 Piprlulu 用于违反公序良俗、色情/暴力等内容哦。

3. 给 GitHub 开发者的小提醒

如果你 Fork 了 Pier 仓库：

  • 完全不用删除：你可以直接保留仓库里的 Piprlulu 资源，比如 README 那些可爱的插画。
  • 谢谢你！：请你在你的 README 第一行至首屏显眼位置 里贴上这段文字：

    本仓库的插画 Piprlulu 属于 @steve372a，其形象受到保护。可以参见 Piprlulu使用规则

  • 豁免情况：如果你不想贴上这段文字，请你移除所有 Piprlulu 原版资源。
  • 期待合作：如果你遵守规则并有精彩的改进，欢迎联系 Sanakaprix，我们很乐意在项目中推荐你的 Fork 分支！

4. 关于商用与分发

  • 禁止商业化：未经许可，严禁将 Piprlulu 用于任何盈利行为。
  • 原版资源保护：请勿直接复制原版资源进行分发，或使用原图进行 AI 模型训练。

感谢你对 Piprlulu 家族的尊重与爱护！
如果有任何疑问或想申请商用授权，请随时联系 Sanakaprix 噢。
