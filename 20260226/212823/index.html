🚩 Sanaka 3.21 考试弱项精准唤醒手册

考试倒计时：23 天
目标：750 分（150 语文 + 150 数学 + 450 计算机职业技能）
策略：保住计算机实操优势，暴力唤醒数学公式，死记 MySQL 核心动词。

一、 计算机职业技能（450分：绝对重灾区/抢分区）

[cite_start]考纲明确：全程纸质笔试，题型为单选、多选、判断。无上机，考文字描述和原理。 [cite: 5, 25, 27, 29]

1. MySQL 数据库（占比 20% + 20%） ★

【弱项：DML 数据操作与 DDL 结构操作混淆】

  • [cite_start]UPDATE：修改表中的数据内容（如改分数）。 [cite: 107]
  • [cite_start]ALTER：修改表结构（如加减列、改字段名）。 [cite: 105]
  • [cite_start]DROP：彻底删除表结构（连壳带肉一起消失）。 [cite: 105]
  • [cite_start]DELETE：只删除数据记录（保留表的结构）。 [cite: 107]
  • [cite_start]主键 (Primary Key)：唯一标识每一行，不能为空，一张表只能有一个。 [cite: 18]

2. Office 2010 选项卡定位（弱项针对）

【弱项：Word 分栏、页眉页脚位置记混】

  • [cite_start]Word 插入选项卡：页眉、页脚、页码、艺术字、图表。 [cite: 81, 79]
  • [cite_start]Word 页面布局选项卡：分栏、分节符、页边距、纸张方向。 [cite: 81, 77]
  • [cite_start]Excel 引用：$A$1 是绝对引用（复制公式时不改变），A1 是相对引用。 [cite: 87]
  • [cite_start]PPT 交互设置：动画针对对象（飞入、淡出）；切换针对幻灯片页面之间。 [cite: 100]

3. 基础知识（你的优势，防止细节丢分）

  • [cite_start]存储换算：$1TB = 1024GB$，$1GB = 1024MB$。 [cite: 14]
  • [cite_start]网络分层：网络层负责 IP 编址；光纤抗干扰最强。 [cite: 70, 71, 68]
  • [cite_start]操作系统：制作系统启动 U 盘通常涉及磁盘分区与格式化（NTFS/FAT32）。 [cite: 52, 53]

二、 数学（150分：公式暴力提取）

【弱项：公式完全断片，计算符号混淆】

1. 集合与函数

  • 集合符号：
    • $\cap$ (交集)：取公共部分（相同的元素）。
    • $\cup$ (并集)：取所有元素合并。
  • 奇偶性：
    • 偶函数：$f(-x) = f(x)$（图像关于 $y$ 轴对称，值不变）。
    • 奇函数：$f(-x) = -f(x)$（图像关于原点对称，值变号）。

2. 数列与几何（必背公式）

  • 等差数列通项：$a_n = a_1 + (n-1)d$。
  • 圆的标准方程：$(x - a)^2 + (y - b)^2 = r^2$。
    • 圆心坐标：$(a, b)$，注意符号。例：$(x-1)^2 + (y+2)^2 = 4$，圆心是 $(1, -2)$。
  • 一次函数图像：$y = kx + b$。
    • $k < 0, b > 0$ 时，经过一、二、四象限。

三、 语文（150分：专项防丢分）

【弱项：谦敬辞场景判断错误、病句搭配不当】

1. 谦敬辞对照表

  • 谦辞 (称呼自己)：拙作（文章）、家父（父亲）、鄙见（见解）。
  • 敬辞 (称呼别人)：令爱（女儿）、惠顾（光临）、垂询（询问）。

2. 病句雷区

  • 主谓不当：抽象主语不能接具体的动作。例：“品质”不能“浮现”。
  • 语序不当：多项定语或状语顺序混乱；递进关系（不仅…更…）逻辑颠倒。

  2. 拒绝空题：纸质笔试不倒扣分，数学不会就带入选项倒推，计算机不会就靠第一直觉。
